1. 개요 (https://hunkim.github.io/ml)  - 머신러닝 교육자료
===============
AI < 머신러닝 < 딥러닝

(1) AI
    - 머신러닝, 컴퓨터 비전, 자연어 처리, 로봇 공학 등의 그와 관련된 모든 주제를 포함하는 개념
    
(2) Machine Leaning
    - 더 많은 데이터 축적을 통해 성능을 개선할 수 있도록 하는 다양한 알고리즘과 방법론
    - 데이터를 기반으로 모델을 자동으로 생성하는 기술
    - 실세계의 복잡한 데이터로부터 규칙과 패턴을 발견하여 미래를 예측하는 기술
    
(3) 통계학(엄격하다) -> 데이터 마이닝 = 머신러닝(자유롭다)
    - 머신러닝은 훈련 데이터를 통해 학습된 알려진 속성을 기반으로 한 예측에 중점
    - 데이터 마이닝은 데이터의 미처 몰랐던 속성을 발견하는 것에 집중
    - 데이터 마이닝에서는 머신러닝이 필수는 아니지만, 머신러닝에서는 데이터 마이닝이 필수이다.

(4) 통계학 vs 머신러닝
    1) Data
        - 통계학 : 전체 중에서 일부를 추출하여 추정
        - 머신러닝 : 전체를 가지고 전체를 추정
    
    2) 접근 방법론
        - 통계학 : 이론 중심 -> 연역법
        - 머신러닝 : 현실 중심 -> 귀납법
        
    3) 목적
        - 통계학 : small data -> 사회/여론 조사
        - 머신러닝 : big data -> 현상 예측
        
    
2. 머신러닝의 종류
=======================
(1) 지도 학습(교사 학습) : Supervised Learning
    1) 회귀(Regression)
        - 선형 회귀(Linear Regression)
        - Ridge Regression, Lasso Regression, Elasticnet Regression
    
    2) 분류(Classification) - 따로 설명이 없으면 이진 분류 (A냐 아니냐)
        - Logistic Regression
        - KNN : 근접거리를 기준으로하는 최소 근접 알고리즘
        - Decision Tree
        - SVM
        - Naive Bayes
        - Neural Network
        - ...

(2) 비지도 학습(비교사 학습) : Unsupervised Learning
    - K-means

(3) 강화 학습 : Reinforcement Learning
    - 완전한 답을 제공하지 않는다.
    - 보상 시스템 적용
    - 자체적으로 데이터를 생성
    - 대표적인 예 : 알파고
